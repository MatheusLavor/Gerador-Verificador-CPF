# 🪪 Gerador e Verificador de CPF

Projeto desenvolvido em **Python** com o objetivo de praticar lógica de programação, manipulação de listas, estruturas de repetição e validação de dados através do algoritmo oficial de verificação de CPF.

Este projeto faz parte do meu processo de aprendizado e continuará recebendo melhorias ao longo do tempo.

---

## 🚀 Funcionalidades

- ✅ Verificar se um CPF é válido
- ✅ Gerar um CPF válido a partir dos 9 primeiros dígitos informados
- ✅ Remoção automática de caracteres como "." e "-"
- ✅ Menu interativo no terminal

---

## 🛠️ Tecnologias utilizadas

- Python 3
- Módulo `re`
- Módulo `os`
- Módulo `time`
